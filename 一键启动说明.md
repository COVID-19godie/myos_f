# ZMG Cloud OS 一键启动说明

## 🚀 快速开始

### 方法一：快速启动（推荐新手）
双击运行 `quick_start.bat`
- 自动检测并初始化后端环境
- 自动安装前端依赖（如需要）
- 同时启动前后端服务

### 方法二：完整控制启动
双击运行 `start_all.bat`
- 提供多种启动选项
- 可选择单独启动前后端
- 支持依赖安装和完整设置

### 方法三：手动分步启动

#### 后端启动：
```cmd
cd d:\MyOS\zmg_backend
start.bat
```

#### 前端启动：
```cmd
cd d:\MyOS\frontend
npm install    # 首次运行需要
npm run dev
```

## 📋 启动脚本说明

### 主要文件
- `start_all.bat` - 完整功能启动器（推荐）
- `quick_start.bat` - 快速启动器（新手友好）
- `frontend\install_deps.bat` - 前端依赖安装器
- `zmg_backend\start.bat` - 后端启动器

### 访问地址
- **后端API**: http://127.0.0.1:8000
- **前端界面**: http://localhost:5173
- **API文档**: http://127.0.0.1:8000/api/docs/
- **管理后台**: http://127.0.0.1:8000/admin/

### 日志文件
- `logs\backend.log` - 后端运行日志
- `logs\frontend.log` - 前端运行日志

## ⚠️ 常见问题

### 1. Node.js未安装
错误信息：`Node.js未安装或未添加到环境变量`
解决方案：从 https://nodejs.org/ 下载安装Node.js

### 2. Python未安装  
错误信息：`Python未安装或未添加到环境变量`
解决方案：从 https://python.org/ 下载安装Python

### 3. 端口占用
如果端口被占用，可以修改：
- 后端：修改 `zmg_backend\start.bat` 中的端口
- 前端：修改 `frontend\vite.config.js` 中的端口

### 4. 依赖安装慢
可以使用国内镜像：
```cmd
npm config set registry https://registry.npmmirror.com
```

## 🛑 停止服务
- 关闭对应的命令行窗口
- 或按 `Ctrl+C` 停止当前服务

## 📞 技术支持
如遇问题，请查看相应的日志文件或重新运行启动器。
